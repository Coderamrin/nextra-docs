# Create Product on Stripe Dashboard

So, we’ve added the Secret key to the .env file. Now let’s go ahead and see how you can add a product from the Stripe Dashboard.

1. Go to the **Products** tab.

![stripe](https://nextjstemplates.com/docs/stripe3.png)

2. Then click on the **Add a Product** button

![stripe](https://nextjstemplates.com/docs/stripe4.png)

3. Now add all the details about the product. Then click on the **Save Product** button to save it.

![stripe](https://nextjstemplates.com/docs/stripe5.png)

After you create the Product you'll see them on the dashboard

![stripe](https://nextjstemplates.com/docs/stripe6.png)
